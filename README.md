# Projeto de Análise de Filmes

Este projeto foi desenvolvido em Python para praticar conceitos básicos de programação e análise de dados.

## Funcionalidades

- Listagem de filmes e notas
- Identificação do melhor filme
- Identificação do pior filme
- Cálculo da média das notas
- Quantidade de filmes com nota acima de 9
- Quantidade de filmes com nota abaixo de 7
- Exibição dos filmes acima da média
- Exibição dos filmes abaixo da média

## Tecnologias utilizadas

- Python
- Jupyter Notebook
- Visual Studio Code

## Objetivo

Praticar estruturas fundamentais da linguagem Python, como:

- Variáveis
- Listas
- Estruturas de repetição (`for`)
- Condicionais (`if`)
- Contadores
- Acumuladores
- Manipulação de dados
